Pour utiliser mydrive vous devez : 

I) Télécharger le projet à l'adresse https://github.com/VCROMBECQUE/mydrive.git et le placer dans le dossier "www" de wamp (C:\wamp64\www\mydrive\...)

II) Ouvrir wamp  => créer une base de donnée "my_drive" 
                => importer le fichier "my_drive.sql" qui se trouve à la racine du projet

III) Ouvrer votre navigateur et aller à l'adresse "localhost/mydrive"
